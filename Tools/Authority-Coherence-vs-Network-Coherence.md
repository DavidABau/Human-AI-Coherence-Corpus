# Authority Coherence vs Network Coherence

*A Simple Orientation Tool*

This tool helps quickly recognise:

👉 *How meaning is being stabilised in a situation*

Many modern conflicts are not about truth itself, but about **which coherence strategy is being used**.

---

## The Two Core Strategies

### 🏛 Authority Coherence

Meaning is stabilised by:

* hierarchy
* role legitimacy
* tradition
* doctrine
* recognised expertise

Core question:

> Who determines what is true or right?

Stability comes from:

➡️ Trust in structure

---

### 🌐 Network Coherence

Meaning is stabilised by:

* shared information
* distributed input
* feedback loops
* collective sensemaking
* adaptive models

Core question:

> What emerges as true through interaction?

Stability comes from:

➡️ Trust in process

---

## Quick Comparison

|                     | Authority Coherence | Network Coherence |
| ------------------- | ------------------- | ----------------- |
| Source of stability | Structure           | Interaction       |
| Truth comes from    | Position            | Pattern           |
| Validation method   | Credential          | Convergence       |
| Decision logic      | Top-down            | Emergent          |
| Change style        | Reform              | Iteration         |
| Risk                | Rigidity            | Chaos             |
| Strength            | Clarity             | Adaptability      |

---

## Observable Language Signals

### Authority-style language

* “Experts say…”
* “That’s not how it’s done.”
* “This is the correct interpretation.”
* “Who approved this?”
* “We need leadership.”

Underlying logic:

👉 Stability through legitimacy

---

### Network-style language

* “What are people seeing?”
* “Let’s test it.”
* “Does the data support this?”
* “What’s the pattern here?”
* “Let’s compare inputs.”

Underlying logic:

👉 Stability through feedback

---

## Where Conflict Appears

Most disagreements are actually about:

➡️ How meaning should be stabilised

| Situation | Authority Response      | Network Response          |
| --------- | ----------------------- | ------------------------- |
| Science   | Institutional consensus | Open replication          |
| Parenting | “Because I said so”     | Collaborative negotiation |
| Work      | Chain of command        | Team iteration            |
| Religion  | Sacred text             | Interpretive dialogue     |
| Politics  | Leadership mandate      | Participatory systems     |

People may appear to argue about facts…

…but often disagree about:

👉 the *method of coherence*

---

## Failure Modes

### Authority Failure → Oppression

* rigidity
* gatekeeping
* stagnation
* enforced conformity

---

### Network Failure → Drift

* fragmentation
* relativism
* noise
* paralysis

Healthy systems oscillate between both.

---

## Practical Use

When tension appears, ask:

1. What coherence strategy is active here?
2. Is it appropriate for the situation?
3. Is there a mismatch?

---

## Example: Relational Mismatch

Peg’s brother responded:

> “She never taught me.”

This reflects:

➡️ Authority relevance filtering

Meaning evaluated by:

> Role relation

Peg was operating in:

➡️ Network / relational coherence

Meaning came from:

> Shared experience

Mismatch = emotional drop.
