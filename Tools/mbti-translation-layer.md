# Optional MBTI Translation Layer
## (Non-Binding Correlation)

### Purpose

This section exists solely as an accessibility bridge.

Some users are familiar with MBTI and may find it helpful as a translation aid when first encountering Alignment Expression Channels.

This layer is:

Optional  
Non-deterministic  
Interpretive  

It does not form part of the core model.

---

## Core Safeguard

MBTI does not determine alignment.

It may influence:

Behavioural expression style.

Channels remain:

Contextual  
Adaptive  
Situational  

They are not fixed personality traits.

---

## Possible Tendencies (Soft Correlation)

Some cognitive styles may show tendencies toward particular expression channels.

| Cognitive Lean | Possible Channel Tendency |
|----------------|---------------------------|
| ISTJ | Duty |
| ISFP | Authentic |
| ISTP | Practical |
| ENFJ | Interpersonal |
| INTJ | Strategic |
| ENFP | Evolutionary |
| ESFP | Experiential |

These are:

Heuristic observations  
—not identity mappings.

---

## Important Distinction

MBTI describes:

Cognitive preference.

Alignment Expression Channels describe:

Behavioural enactment of value.

The two operate at different levels.

---

## Misuse Prevention

This section should not be used to:

- Assign moral style to personality type
- Predict virtue or alignment
- Categorise individuals

Its role is:

Interpretive translation only.

---

## One-Line Summary

MBTI may influence how alignment is expressed —  
but it does not define alignment.
