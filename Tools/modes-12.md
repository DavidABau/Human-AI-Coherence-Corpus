# The 12 Modes

An expanded version of the 5 Modes for when more nuance is needed.

Use only if the 5 Modes feel insufficient.

## Regulation Layer
Regulation  
Containment  

## Connection Layer
Bonding  
Validation  

## Expression Layer
Narrative  
Meaning-making  

## Understanding Layer
Diagnostic  
Sense-making  

## Agency Layer
Boundary-setting  
Action / Agency  

## Play Layer
Play / Humour  
Paradox  

## Purpose

The 12 Modes help distinguish:

- comfort vs structure  
- validation vs meaning  
- humour vs avoidance  
- advice vs agency  

They are not required for everyday use.

Start simple. Expand only when needed.
