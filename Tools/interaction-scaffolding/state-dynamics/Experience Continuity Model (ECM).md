# Experience Continuity Model (ECM)

**Purpose:**
Provide a simple, structural way to understand consciousness, anesthesia, blackouts, and time perception without relying on metaphysical assumptions.

---

## Core Principle

> **Perceived time and experience require continuous, integrated, memory-forming signal.**

Or tighter:

> **No continuity → no memory → no time.**

---

## Functional Definition

Instead of treating consciousness as a “thing,” ECM treats it as a **system function**:

> **Experience = integrated signal over time + memory persistence**

---

## The Four Conditions

For experience to exist (and be reportable), four conditions must be met:

1. **Signal**
   Raw activity (neural or system-level dynamics)

2. **Integration**
   Signals are unified into a coherent state (not fragmented noise)

3. **Continuity**
   The state persists and updates across time

4. **Persistence (Memory)**
   Some of the state is retained for reconstruction

---

## System Loop

```
Signal → Integration → Experience → Memory → Feedback → Continuity
```

This loop produces:

* stable experience
* time perception
* reportability

---

## Failure Modes

### 1. Full Continuity (Waking)

* Signal ✓
* Integration ✓
* Continuity ✓
* Memory ✓

→ Stable experience
→ Linear time perception

---

### 2. Distorted Continuity (Dreaming)

* Signal ✓
* Integration ~
* Continuity ~
* Memory ~

→ Experience present but unstable
→ Time distorted

---

### 3. Broken Continuity (Anesthesia)

* Signal ✓
* Integration ✗
* Continuity ✗
* Memory ✗

→ No reconstructable experience
→ Time collapses to zero

> **Interpretation:**
> The system cannot register the interval, so it is later experienced as “nothing happened.”

---

### 4. Partial Persistence Failure (Blackout)

* Signal ✓
* Integration ✓
* Continuity ✓
* Memory ✗

→ Experience may occur
→ Not retained
→ Time appears missing in hindsight

---

## Key Insight

> **Time perception is a reconstruction problem, not a direct measurement.**

The system infers duration from:

* continuity of experience
* number of state transitions
* memory checkpoints

Remove these → duration = 0 (subjectively)

---

## Anesthesia (Operational Explanation)

Anesthesia does not need to “turn off consciousness.”

It disrupts:

* integration (network coordination)
* continuity (stable state propagation)
* persistence (memory formation)

Result:

> **No integrated signal + no persistence → no reconstructable interval**

---

## Practical Diagnostic Use

To analyse any state, ask:

* Is signal present?
* Is it integrated?
* Is it continuous?
* Is it being stored?

This avoids:

* metaphysical assumptions
* vague “conscious vs unconscious” framing

> **Experience = integration × continuity × memory**
