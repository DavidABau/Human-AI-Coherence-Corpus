# State Dynamics Starter Bundle

**Purpose:** Provide a minimal, practical system for understanding and steering internal states in real time.

---

## Overview

This bundle contains three complementary tools:

1. **ECM — Experience Continuity Model**
   → explains how experience and time are formed

2. **SSM — Signal Steering Model**
   → explains how state direction is selected

3. **LCM — Load & Collapse Model**
   → explains when control fails and why

---

## Core Loop

```text
Continuity (ECM) → Detection (SSM) → Steering (SSM) → Reinforcement (SSM) → Baseline (SSM)
                                 ↑
                              Load (LCM)
```

## README.md

### What this is

A simple system for:

* understanding internal states
* detecting them early
* steering them when possible
* recognising when steering is not possible

---

### Core Principles

> **No continuity → no time**
> **No detection → no choice**
> **High load → no control**

---

### How to Use

1. Notice state change
2. Check load level
3. If possible → steer early
4. If not → reduce load first

---

### Design Constraints

* No metaphysics
* No clinical claims
* No personality modelling
* Purely structural and practical
